#冒泡排序

```python
#冒泡排序
def BubbleSort(list):
    for i in range(len(list)-1,-1,-1):
        for j in range(i):
            if list[j] > list[j+1]:
                temp = list[j+1]
                list[j+1] = list[j]
                list[j] = temp
    return list
list = [3,2,1,4]
print(BubbleSort(list))
```
