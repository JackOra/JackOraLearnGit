## 二分查找

### 二分查找的前提

1. 目标函数的单调性(**有序**)(单调递增或者递减)
2. 存在上下界 (bounded)
3. 能够通过索引进行访问(index accessible)

#### 代码模版1

```python
low, high = 0, len(nums) - 1
while low <= high:
  mid = low + (high - low) // 2
  if nums[mid] == target:
    break or return result
  elif nums[mid] < target:
    low = mid + 1
  else:
    high = mid - 1
```

#### 代码模版2

```python
left, right = 0, len(nums) - 1
while left < right:
  mid = left + (right - left) // 2
  if nums[mid] == target:
    return mid
  elif nums[mid] < target:
    left = mid + 1
    else:
      right = mid
```
