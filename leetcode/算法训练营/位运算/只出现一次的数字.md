#### [只出现一次的数字](https://leetcode-cn.com/problems/single-number/)

> 给定一个**非空**整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

```java
class Solution {
    public int singleNumber(int[] nums) {
        int x = 0;
        for (int num:nums){
            //异或来实现
            x ^= num;
        }
        return x;
    }
}
```

1. 交换律：a ^ b ^ c <=> a ^ c ^ b
2. 任何数于0异或为任何数 0 ^ n => n
3. 相同的数异或为0: n ^ n => 0
